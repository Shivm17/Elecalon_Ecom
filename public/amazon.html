Complete DVA-C02 Exam Blueprint

01. Deployment and Provisioning (22% of Exam) 8 Hrs

CloudFormation, Elastic Beanstalk, EC2 deployment strategies, Parameter Store, and handling application configurations securely.

02. Security (26% of Exam) 13 Hrs

IAM roles/policies, Resource Access Control (S3 bucket policies, security groups), Secrets Manager, and application security best practices.

03. Development with AWS Services (30% of Exam) 15 Hrs

CloudWatch Logs, Metrics, Alarms, X-Ray for distributed tracing, and implementing logging and debugging across services.

05. Testing (8% of Exam) 4 Hrs

Testing strategies for serverless applications, performing A/B tests with Lambda versions, and automated rollbacks with CodeDeploy.
