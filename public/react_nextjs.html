In-Depth Modules

01. React Fundamentals: Components & Props 10 Lectures / 5 Hrs

JSX, functional vs. class components, prop drilling solutions, and setting up your Vite/Next.js environment.

02. State, Hooks, and Component Lifecycle 14 Lectures / 7 Hrs

The power of `useState`, side effects with `useEffect`, performance optimization with `useMemo` and `useCallback`, and the Context API.

03. Next.js Routing and Data Fetching (App Router) 18 Lectures / 9 Hrs

Nested layouts, dynamic routes, loading states, error boundaries, and fetching data using server and client patterns.

04. Fullstack Development: Server Actions & APIs 12 Lectures / 6 Hrs

Implementing Server Actions for mutations, building secure API routes, and connecting to a database (using Prisma ORM).

05. Authentication, Testing, and Deployment 11 Lectures / 5.5 Hrs

Setting up session management, writing component tests with Jest/RTL, and deploying projects to Vercel and Netlify.
