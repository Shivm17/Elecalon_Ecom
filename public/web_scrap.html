About the Role

As a Web Scraping Specialist at Elecalon, you will be responsible for extracting valuable data from various websites using advanced web scraping tools and organizing it in structured formats for business analysis and decision making.

Key Responsibilities:

  • Perform efficient data scraping tasks from assigned websites using various tools and techniques
  • Organize, clean and validate scraped data for accurate reporting
  • Ensure compliance with data policies and website terms of service
  • Maintain detailed records of data sources and scraping processes
  • Troubleshoot and resolve scraping issues promptly
  • Collaborate with the data analysis team to understand requirements
